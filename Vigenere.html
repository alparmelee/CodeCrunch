(5 pts) Encrypt the plaintext "BERRYWATER" using the Vigenere Cipher with the keyword "CUBE".

Answer: DYSVAQBXGL

To encrypt "BERRYWATER" with the keyword "CUBE", you repeat the keyword to match the length of the plaintext: CUBECUBECU. The encryption formula is Ci = (Pi + Ki) mod 26. B(1) + C(2) = 3 = D E(4) + U(20) = 24 = Y R(17) + B(1) = 18 = S R(17) + E(4) = 21 = V Y(24) + C(2) = 26 = 0 = A W(22) + U(20) = 42 = 16 = Q A(0) + B(1) = 1 = B T(19) + E(4) = 23 = X E(4) + C(2) = 6 = G R(17) + U(20) = 37 = 11 = L The ciphertext is DYSVAQBXGL.

(10 pts) The following ciphertext was encrypted using the Vigenere cipher with the keyword "SKY": KELSXBEYMF. What was the original plaintext?

Answer: SUNANDMOON

To decrypt "KELSXBEYMF" with the keyword "SKY", you repeat the keyword: SKYSKYSKY. The decryption formula is Pi = (Ci - Ki) mod 26. K(10) - S(18) = -8 = 18 = S E(4) - K(10) = -6 = 20 = U L(11) - Y(24) = -13 = 13 = N S(18) - S(18) = 0 = A X(23) - K(10) = 13 = N B(1) - Y(24) = -23 = 3 = D E(4) - S(18) = -14 = 12 = M Y(24) - K(10) = 14 = O M(12) - Y(24) = -12 = 14 = O F(5) - S(18) = -13 = 13 = N The plaintext is SUNANDMOON.
