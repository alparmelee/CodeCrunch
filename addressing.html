üî¢ Network Addressing

üîë
MAC Address: A hardware address unique to each network interface card (NIC). Consists of 48 bits displayed in hexadecimal notation. The first half identifies the manufacturer, while the second half is a unique device identifier.
üåê
IP Address: A 32-bit (IPv4) numeric address that identifies a device on a network. Consists of four octets (numbers 0-255) separated by dots, divided into network and host portions determined by the subnet mask.
üö™
Port Number: A number that addresses the receiving application in the Transport layer header. Well-known ports (0-1023), registered ports (1024-49151), and dynamic ports (49152-65535).
