IC & MIC Questions

Fall 2021 Quiz #1

(8 pts) What is the index of coincidence of the following set of letters: 10 As, 50 Bs, 50 Cs, and 40 Ds? For full credit, please express your answer as a fraction in lowest terms.

Answer: 131/447 ≈ 0.29306

The total number of letters, N, is 10+50+50+40=150. The formula for the index of coincidence, IC, is: IC = Sum of [ f(i) * (f(i) - 1) ] / [ N * (N - 1) ] The sum of f(i) * (f(i) - 1) is: A: 10(10-1) = 90 B: 50(50-1) = 2450 C: 50(50-1) = 2450 D: 40(40-1) = 1560 Sum = 90 + 2450 + 2450 + 1560 = 6550 The denominator is N(N-1) = 150(149) = 22350. IC = 6550 / 22350 To reduce the fraction, divide both the numerator and denominator by their greatest common divisor, 50. IC = (6550 / 50) / (22350 / 50) = 131 / 447.

(6 pts) What is the index of coincidence of the following set of letters? 10 As, 40 Bs, 20 Cs, 80 Ds. Leave your answer as a fraction, in reduced form.

Answer: 167/447 ≈ 0.37360

The total number of letters, N, is 10+40+20+80=150. The formula for the index of coincidence, IC, is: IC = Sum of [ f(i) * (f(i) - 1) ] / [ N * (N - 1) ] The sum of f(i) * (f(i) - 1) is: A: 10(10-1) = 90 B: 40(40-1) = 1560 C: 20(20-1) = 380 D: 80(80-1) = 6320 Sum = 90 + 1560 + 380 + 6320 = 8350 The denominator is N(N-1) = 150(149) = 22350. IC = 8350 / 22350 To reduce the fraction, divide both the numerator and denominator by their greatest common divisor, 50. IC = (8350 / 50) / (22350 / 50) = 167 / 447.
Fall 2020 Quiz #1

(10 pts) Consider a set of letters with 1 A, 2 Bs, 3 Cs, 4 Ds, ..., and 26 Zs. What is the index of coincidence of this set of letters? Express your answer as a fraction in lowest terms.

Answer: 1/21 ≈ 0.04762

The total number of letters is the sum of the first 26 integers: N = (26 * 27) / 2 = 351. The numerator of the IC formula is the sum of n(i) * (n(i) - 1), where n(i) is the count of each letter. Numerator = Sum from i=1 to 26 of [ i * (i-1) ] Using the hint from the original document: [(n-1) * n * (n+1)] / 3 for n=26. Numerator = [(26-1) * 26 * (26+1)] / 3 = (25 * 26 * 27) / 3 = 5850. The denominator of the IC formula is N * (N-1) = 351 * 350 = 122850. IC = 5850 / 122850. To reduce the fraction, divide both the numerator and denominator by their greatest common divisor, 5850. IC = (5850 / 5850) / (122850 / 5850) = 1 / 21.
