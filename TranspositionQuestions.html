(8 pts) The following ciphertext has been encrypted via column transposition, using the keyword "TENNESSEE". What is the decrypted plaintext?

EFCEV DMGEM VENTD RENDE IIALO HMOIL RMELL EEABE AATNE MRTIA RNN

Answer: TELLMEANDIFORGETTEACHMEANDIREMEMBERINVOLVEMEANDILEARN

Keyword: TENNESSEE (9 letters) Ranks → [T=9, E1=1, N1=5, N2=6, E2=2, S1=7, S2=8, E3=3, E4=4] Order of reading: [1,4,7,8,2,3,5,6,0] Ciphertext length = 53 → 9 columns → 6 rows (one short) Column lengths: [6,6,6,6,6,6,6,6,5] Split ciphertext: col1 = EFCEVD col4 = MGEMVE col7 = NTDREN col8 = DEIIA col2 = LOHMOI col3 = LRMELL col5 = EEABEA col6 = ATNEMR col0 = TIARNN Rearranged grid: T E L L M E A N D I F O R G E T T E A C H M E A N D I R E M E M B E R I N V O L V E M E A N D I L E A R N Reading row-wise: TELLMEANDIFORGETTEACHMEANDIREMEMBERINVOLVEMEANDILEARN
Fall 2021 Quiz #2

(15 pts) Complete the function in C below which takes in a plaintext string, an integer array storing the permutation, and the length of that array and returns the ciphertext.

Answer: The completed C function.

The function takes the plaintext and encrypts it block by block using the permutation array.
#include <string.h> #include <stdlib.h> char* permEncrypt(char* plain, int* perm, int pLen) { int msgLen = strlen(plain); char* cipher = (char*)malloc((msgLen + 1) * sizeof(char)); if (cipher == NULL) return NULL; cipher[msgLen] = '\0'; for (int i = 0; i < msgLen; i += pLen) { for (int j = 0; j < pLen; ++j) { cipher[i + j] = plain[i + perm[j]]; } } return cipher; }
