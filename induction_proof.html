Understanding Mathematical Induction

Mathematical induction is a powerful proof technique used to establish that a statement is true for all natural numbers. Think of it like climbing a ladder: first, you make sure you can reach the first rung, then you prove that if you can reach any rung, you can always reach the next one.

Step 1: Base Case

Prove that the statement P(n) is true for the smallest value of n (usually n = 0 or n = 1).

This is like confirming you can climb onto the first rung of the ladder.

Step 2: Inductive Hypothesis

Assume that P(k) is true for some arbitrary k ≥ initial value.

This means we assume we've already reached the k-th rung of the ladder.

Step 3: Inductive Step

Prove that if P(k) is true, then P(k+1) is also true.

This demonstrates that if you can reach the k-th rung, you can always climb to the next rung.

Step 4: Conclusion

Conclude that P(n) is true for all valid values of n by the principle of mathematical induction.

Since you can reach the first rung, and from any rung you can reach the next one, you can climb the entire ladder!

Example 1: Sum of the First n Odd Numbers

Prove: 1 + 3 + 5 + ... + (2n-1) = n² for all n ≥ 1

Step 1: Base Case (n = 1)

For n = 1: The left side is just the first odd number: 1

The right side is 1² = 1

Since 1 = 1, the base case is true.

Step 2: Inductive Hypothesis

Assume the statement is true for n = k:

1 + 3 + 5 + ... + (2k-1) = k²

Step 3: Inductive Step

We need to prove it's true for n = k+1:

1 + 3 + 5 + ... + (2k-1) + (2(k+1)-1) = (k+1)²

Using our inductive hypothesis:

k² + (2k+1) = (k+1)²

k² + 2k + 1 = k² + 2k + 1

This is true! So P(k+1) is true when P(k) is true.

Step 4: Conclusion

By mathematical induction, the formula is true for all n ≥ 1.

Example 2: Divisibility by 6

Prove: n³ - n is divisible by 6 for all n ≥ 1

Step 1: Base Case (n = 1)

For n = 1: n³ - n = 1³ - 1 = 1 - 1 = 0

0 is divisible by 6, so the base case is true.

Step 2: Inductive Hypothesis

Assume the statement is true for n = k:

k³ - k is divisible by 6

This means k³ - k = 6m for some integer m.

Step 3: Inductive Step

We need to prove it's true for n = k+1:

(k+1)³ - (k+1) is divisible by 6

(k+1)³ - (k+1)

= k³ + 3k² + 3k + 1 - k - 1

= k³ + 3k² + 2k

= (k³ - k) + 3k² + 3k

= 6m + 3k(k + 1)

Since k and k+1 are consecutive integers, one of them must be even, so k(k+1) is always even.

This means 3k(k+1) is always divisible by 6.

Therefore, 6m + 3k(k+1) is divisible by 6.

Step 4: Conclusion

By mathematical induction, n³ - n is divisible by 6 for all n ≥ 1.
